### 判断类型
 * 1、typeof
  ```javascript
  typeof 1 // number
  typeof '' // string
  typeof true // boolean
  typeof [] // object
  typeof {} // object
  typeof undefined // undefined
  typeof null // object
  typeof function(){} // function
  ```
 * 2、instanceof
 * 3、constructor
 * 4、Object.prototype.toString.call()
