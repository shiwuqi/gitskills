### 判断类型
 * 1、typeof
  >>typeof 1 // number <br />
  >>typeof '' // string <br />
  >>typeof true // boolean <br />
  >>typeof [] // object <br />
  >>typeof {} // object <br />
  >>typeof undefined // undefined <br />
  >>typeof null // object <br />
  >>typeof function(){} // function <br />
